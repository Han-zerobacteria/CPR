# CPR(Cloth Personal Rescue)

**CPR (Cloth Personal Rescue)**는
옷장 안에 입고 싶지만 어떻게 입어야 이쁘게 입을 수 있을지 모르겠는 옷의 코디를 의뢰할 수 있도록 도와주는 서비스로써
심폐소생술의 약자인 CPR을 Cloth Personal Rescue로 바꿔
사용자의 패션을 개인 맞춤으로 구조한다는 의미를 담고 있습니다.

✅ 주요 기능 구조

📦 옷장 관리
이미지 업로드 (모바일 사진 or 촬영)

옷 분류 (셔츠/바지/치마 등 자동 태깅)

🧠 AI

AI 기반 누끼 제거를 통하여
사용자 편의성 개선

🧾 ootd 공유 피드 & 코디 의뢰 게시판

자신의 ootd를 공유하고 다른 사람의 코디를
살펴보며 마음에 드는 사람에게 코디를 의뢰합니다.
